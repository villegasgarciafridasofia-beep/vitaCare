# Descripción

**VitaCare** es una aplicación móvil desarrollada para ayudar a los usuarios a administrar su información médica y llevar un control de sus medicamentos, citas y recordatorios de salud desde un dispositivo móvil.

La aplicación permite organizar información importante relacionada con el cuidado personal, facilitando el seguimiento de tratamientos médicos y mejorando la adherencia a los medicamentos.

---

# Problemática

Muchas personas olvidan tomar sus medicamentos, asistir a sus consultas médicas o llevar un registro adecuado de su información clínica, lo que puede afectar el seguimiento de los tratamientos y su bienestar.

Además, la información médica suele encontrarse dispersa entre recetas, notas o documentos físicos, dificultando su consulta cuando se necesita.

---

# Solución Propuesta

Se desarrolló una aplicación móvil que permite:

- Registrar medicamentos.
- Programar recordatorios.
- Gestionar citas médicas.
- Administrar información personal.
- Consultar el historial de tratamientos.
- Mantener organizada la información médica del usuario.

---

# Arquitectura

La aplicación fue desarrollada siguiendo una arquitectura **MVVM (Model-View-ViewModel)** para separar la lógica de negocio de la interfaz de usuario.

La arquitectura está compuesta por:

- Interfaz desarrollada con Xamarin.Forms.
- Lógica de negocio en C#.
- Base de datos local SQLite.
- Navegación mediante FlyoutPage.

---

# Desarrollo del Proyecto

El desarrollo inició con el análisis de las necesidades relacionadas con la administración de tratamientos médicos y recordatorios.

Posteriormente se diseñó la base de datos para almacenar la información de usuarios, medicamentos y citas médicas.

Se implementó la interfaz móvil utilizando Xamarin.Forms y se desarrollaron los módulos encargados de gestionar la información médica del usuario.

Finalmente se realizaron pruebas funcionales para validar el correcto funcionamiento de los recordatorios, el registro de medicamentos y la administración de citas.
